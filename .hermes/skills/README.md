# Hermes Skills

Project-specific repeated procedures go here.

This directory is intentionally empty in the MVP. Do not pre-fill it with speculative skills.

Create a skill only when:

- The procedure has repeated at least three times.
- Inputs and outputs are clear.
- Verification criteria are clear.
- The procedure is specific enough to this Hermes setup that a generic tool or short rule is insufficient.
- The addition passes the human gate in `policy/automation.md`.

Examples that may become skills later:

- Decision trace writing.
- Wiki ingest.
- Wiki lint.
- Handoff writing.
- Meta budget check.
